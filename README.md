# Sales_Analysis_Dashboard
An interactive Sales &amp; Revenue Analysis Dashboard built using Python (Pandas) and Power BI to analyze sales performance, visualize KPIs, and generate business insights from retail sales
# Sales & Revenue Analysis Dashboard

## Project Overview
This project analyzes sales data using Python and Power BI to generate business insights through interactive dashboards.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Power BI
- Excel/CSV

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Sales

## Visualizations
- Monthly Sales Trend
- Top 10 Products
- Sales by Category
- Sales by Region
- Interactive Slicers

## Business Insights
- Technology category generated the highest sales.
- West region had the highest revenue.
- Sales peaked during November and December.
- A few products contributed significantly to overall sales.
